# Contributing to OpenUI Kit

Thank you for your interest in contributing! 🎉

## How to Contribute

### Reporting Bugs
- Open a GitHub Issue
- Describe the bug clearly
- Include browser/OS information
- Add a screenshot if possible

### Adding New Components
1. Fork the repository
2. Create a new branch: `git checkout -b feature/component-name`
3. Add your component CSS in `css/openui.css`
4. Add demo HTML in `index.html`
5. Add JS interactions in `js/openui.js` (if needed)
6. Update the README component table
7. Open a Pull Request with a clear description

### Code Style
- Use CSS custom properties for colors
- Follow existing naming conventions (BEM-inspired)
- Ensure keyboard accessibility
- Test on Chrome, Firefox, Safari, and Edge

## Testing

We use BrowserStack for cross-browser testing. All components should work on:
- Chrome 80+
- Firefox 78+
- Safari 14+
- Edge 80+
- Mobile browsers (iOS Safari, Android Chrome)

## Questions?

Open an issue with the label `question`. We're happy to help!
